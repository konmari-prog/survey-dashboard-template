セットアップ手順

6ステップで完了。コーディング不要、コピペだけ

1

スプレッドシートを新規作成

sheets.new にアクセスして新しいスプレッドシートを作成。名前を付ける（例：AI講座 アンケート集計）

2

Googleフォームを紐付け

既存のGoogleフォーム（アンケート）を開いて：

フォーム > 回答タブ > スプレッドシートにリンク

「既存のスプレッドシートを選択」> Step 1 で作ったシートを指定

複数フォームがある場合は、全部同じスプレッドシートに紐付けてください

3

Apps Script エディタを開いてスクリプトIDをコピー

スプレッドシートのメニューから：

拡張機能 > Apps Script

左サイドバーの歯車アイコン（プロジェクトの設定） > スクリプトIDをコピー

4

テンプレートを push

ターミナル（コマンドライン）で以下を実行：

git clone https://github.com/konmari-prog/survey-dashboard-template.git
cd survey-dashboard-template/src
cat > .clasp.json << EOF
{
  "scriptId": "ここにStep3のスクリプトIDを貼る",
  "rootDir": "."
}
EOF

clasp push --force

infoclasp が未インストールの場合

npm install -g @google/clasp
clasp login
5

初期セットアップを実行

スプレッドシートをリロードして：

アンケート管理 > 初期セットアップ

フォーム回答シートが自動検出される（何個でもOK）

ヘッダーのキーワードから列を自動マッピング

プロジェクト名を入力 → テーマカラーを選択

Q&Aシートが自動作成される

セットアップ後、メニューから 「トリガーを設定」 も忘れずに実行してね。フォーム送信時にQ&Aシートへ自動取り込みされるようになります

6

WebAppをデプロイ

Apps Script エディタで：

デプロイ > 新しいデプロイ

種類：ウェブアプリ

実行ユーザー：自分

アクセス：全員（URLを知っている人だけがアクセス）

デプロイ後のURLをコピーして、メニュー 「アンケート管理」>「WebアプリURL設定」 に貼り付け

使えるURL

デプロイURLQ&A質問回答集（受講者向け）
URL?page=dashboardダッシュボード（管理者向け）
URL?page=guideセットアップガイド

ヘッダー自動検出のしくみ

フォームの質問名が違っても、キーワードが含まれていれば自動でマッチ

例：3つのフォームで「満足度」の項目名が違う場合

フォーム1 「満足度」 check_circle
フォーム2 「全体の満足度」 check_circle
フォーム3 「満足」 check_circle

「満足」というキーワードが含まれていれば、すべて satisfaction 列としてマッピングされます

データの流れ

集計処理はゼロ。ダッシュボードを開いた瞬間にスプレッドシートを読みに行く

send

フォーム送信

受講者がフォームに回答。スプレッドシートに自動蓄積

open_in_browser

ダッシュボードを開く

URLにアクセスするとGASがスプレッドシートの全データを読み取り

insights

グラフ表示

Chart.jsでリアルタイム描画。毎回最新データ

lightbulb ポイント：事前の集計処理は一切不要。ダッシュボードを開くたびにスプレッドシートから直接データを取得して表示するしくみです

よくある質問

セットアップ中に困ったら

フォームは何個まで紐付けられますか？ expand_more

上限はありません。1つのスプレッドシートに何個でもフォームを紐付けできます。セットアップ時に「タイムスタンプ」列があるシートをすべて自動検出します。

既存のフォーム（すでに回答があるもの）でも使えますか？ expand_more

はい、そのまま使えます。スプレッドシートに紐付ければ、過去の回答データもダッシュボードに反映されます。

ヘッダーにキーワードが入っていない列はどうなりますか？ expand_more

「自由記述」としてダッシュボードの「受講者の声」タブに自動表示されます。無駄になるデータはありません。

Q&Aシートに質問が自動で入らないのですが？ expand_more

メニュー「アンケート管理」>「トリガーを設定」を実行してください。これでフォーム送信時に質問が自動取り込みされます。また、既存の回答を取り込むには「フォーム回答を取り込む」を使ってください。

テーマカラーは後から変更できますか？ expand_more

メニュー「アンケート管理」>「セットアップをやり直す」で再設定できます。Q&Aシートのデータは保持されます。
